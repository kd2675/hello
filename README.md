# Go 언어 학습 및 프리코스 프로젝트 포팅

### Go 언어의 기본 문법과 특성, 프리코스에서 경험한 Java 프로젝트를 Go 언어로 변환한 2주간의 자율 학습 프로젝트.

도전목표:
- go 언어 기초✅
- java > go 변환 : 객체지향 개념✅
- 테스트 & 동시성
- 웹 개발
- 데이터베이스 & 배포

학습 과정:
- 1단계: 기본 문법 학습

- 변수 선언 및 타입 시스템 이해
  - 기본 자료형
  - 연산자 및 제어문
  - 함수 정의 및 호출

- 2단계: 중급 개념
  - 익명 함수 및 클로저
  - 배열, 슬라이스, 맵
  - 패키지 구조 및 모듈 관리

- 3단계: 고급 개념
  - 구조체와 메서드
  - 인터페이스 기반 설계
  - 에러 핸들링

- 4단계: 프로젝트 포팅
  - 계산기
  - 레이싱카
  - 로또

#### go 언어 구조 이해
- 클래스 → 구조체(Struct) + 메서드
- 상속 → 인터페이스 구현 (composition over inheritance)
- 예외(Exception) → 에러 반환 패턴
- 다중 반환값을 통한 에러 처리
- 구조체 리터럴을 활용한 간결한 초기화
- 슬라이스의 동적 크기 조정
- defer를 활용한 리소스 정리

##### 프로젝트 기간: 2024.11.04 - 2024.11.18 (2주)
##### 학습 방식: Go 기초 학습 → 프리코스 프로젝트 포팅

### 소감
프리코스에서 배운 설계 원칙들이 다른 언어에서도 동일하게 적용된다는 것을 체험했습니다.
특히 작은 함수, 명확한 책임, 레이어 분리 같은 원칙은 언어와 관계없이 좋은 코드를 만드는
핵심이라는 것을 다시 한번 깨달았습니다.
Go 언어의 단순함이 처음엔 불편했지만, 프로젝트를 진행하면서 명시적이고 단순한 것이
얼마나 강력한지 알게 되었습니다. 다음 단계로 Go의 진짜 강점인 동시성을 학습할 예정입니다.
